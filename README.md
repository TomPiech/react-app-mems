## General info

This is an application that aims to rate memes and show them on two routes depending on the number of likes and dislikes. If the meme gets more than 5 likes it is moved to route Hot. If in route Hot we add dislike for this meme, it will return to route Regular again.
	
## Technologies
Project is created with:
* React JS
* React-Router-Dom
* Material UI Icon
* CSS
	
## Setup
To run this project, install it locally using npm:

$ choose master branch on github
$ cd mem-app
$ npm install
$ npm start
$ npm install react --save
$ npm install react react-dom --save
